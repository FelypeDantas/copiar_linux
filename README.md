## 🧠 Sobre o Projeto

- Projeto de uma ferramenta de cópia de arquivos inspirada no comando `cp` do Linux.
- Permite copiar arquivos grandes de forma eficiente.
- Possui barra de progresso com velocidade e tempo estimado.
- Suporta cópia recursiva de diretórios.
- Permite retomada de cópias interrompidas.
- Inclui verificação de integridade usando SHA256.
- Pode realizar cópias paralelas usando múltiplas threads.

É um projeto interessante para estudar:

- manipulação de arquivos em Python
- operações de sistema
- processamento paralelo
- construção de ferramentas CLI
